# 黑洞模拟 - 控制指南

完整的键盘和鼠标控制参考手册。

---

## 相机控制

### 鼠标控制

- **左键拖动**: 围绕黑洞环绕相机
  - 水平移动：改变方位角（绕Y轴旋转）
  - 垂直移动：改变仰角（上下旋转）
  - 相机始终聚焦在黑洞中心(0, 0, 0)
  
- **鼠标滚轮**: 缩放
  - 向上滚动：靠近黑洞
  - 向下滚动：远离黑洞
  - 最小距离：1×10¹⁰ 米（~8倍史瓦西半径）
  - 最大距离：1×10¹² 米

- **Shift + 左键点击**: 导出光标位置的射线（Phase 5新功能！）
  - 在屏幕上任意位置点击
  - 自动计算该像素的射线方向
  - 导出完整的测地线路径到CSV文件
  - 文件名：`ray_click_<时间戳>.csv`
  - 非常适合分析特定特征（爱因斯坦环、光子球）

### 键盘控制

#### 物理

- **G**: 切换牛顿引力模拟
  - 开启：物体根据牛顿F=GMm/r²相互影响
  - 关闭：物体不受引力（纯视觉）
  - 默认：关闭

#### 渲染

**HDR曝光（Phase 1）：**
- **E**: 增加曝光（+0.1）
  - 使图像整体更亮
  - 显示微弱的细节
- **Q**: 减少曝光（-0.1）
  - 使图像整体更暗
  - 防止过曝
- **R**: 重置曝光为1.0
  - 返回默认亮度

**泛光效果（Phase 4）：**
- **B**: 切换泛光后处理
  - 开启：亮物体周围发光
  - 关闭：无发光效果
  - 默认：开启
- **+（加号）**: 增加泛光强度（+0.1）
  - 更强的发光效果
  - 更明显的光晕
- **-（减号）**: 减少泛光强度（-0.1）
  - 更微妙的发光
  - 减少光晕

**Kerr度规（Phase 4）：**
- **K**: 切换Kerr度规
  - 开启：旋转黑洞（Kerr度规）
  - 关闭：非旋转黑洞（Schwarzschild度规）
  - 默认：关闭
- **[（左方括号）**: 减少自旋参数（-0.05）
  - 较慢的旋转
  - 范围：0.0（无旋转）到1.0（最大旋转）
- **]（右方括号）**: 增加自旋参数（+0.05）
  - 更快的旋转
  - 显示更强的拖曳效应

#### 科学工具

**射线导出（Phase 4）：**
- **P**: 导出中心射线路径
  - 从相机位置沿视线方向发射一条射线
  - 保存完整的测地线积分
  - 输出：`ray_center.csv`
  - 包含：时间、位置(x,y,z)、速度、能量、角动量

- **C**: 导出锥形模式
  - 从相机位置发射多条射线（锥形）
  - 用于3D可视化测地线束
  - 输出：`cone_pattern.csv`
  - 非常适合在ParaView/Matplotlib中绘制

**CSV文件格式：**
```csv
lambda,x,y,z,vx,vy,vz,r,theta,phi,E,L
0.000000,6.34e+10,0.00e+00,0.00e+00,-1.00,0.00,0.00,6.34e+10,1.57,0.00,1.00,0.00
...
```

#### 可视化模式（Phase 5新功能！）
- **V**: 循环切换可视化模式
  - **模式0 - 正常**：使用Shakura-Sunyaev盘模型的逼真渲染
  - **模式1 - 引力红移**：按红移因子着色
    * 蓝色 = 弱红移（远离黑洞）
    * 黄橙色 = 中等红移
    * 红色 = 极端红移（接近光子球）
  - **模式2 - 积分步数**：显示计算复杂度
    * 蓝色 = 少数步数（快速射线）
    * 红色 = 多步数（复杂轨道）
  - **模式3 - 能量守恒**：验证数值积分器
    * 绿色 = 完美守恒（RK4正常工作）
    * 黄色 = 可接受
    * 红色 = 差（积分问题）
  - **模式4 - Carter常数**：监控Kerr度规守恒
    * 绿色 = 良好守恒（精确测地线）
    * 橙红色 = 漂移（简化方程）

#### 波段（Phase 6新功能！）
- **W**: 循环切换波段
  - **波段0 - 射电**：长波（毫米-厘米），冷外区
    * 红橙黄伪彩色
    * 显示磁场结构（简化）
  - **波段1 - 红外**：热辐射（1-10 μm），温暖尘埃
    * 橙红热特征
    * 中等温度区域
  - **波段2 - 可见光**：可见光（400-700 nm），自然颜色
    * 标准逼真渲染
    * 默认模式
  - **波段3 - X射线**：高能量（0.1-10 nm），热等离子体
    * 蓝紫白颜色
    * 仅最热内区可见
  - **波段4 - 多波段**：复合视图（哈勃调色板）
    * 红色 = 射电（冷），绿色 = 可见光，蓝色 = X射线（热）
    * 彩虹色温度梯度

#### 参数调整（Phase 6新功能！）
- **1**: 减少曝光（精细）-0.01每次按下
- **2**: 增加曝光（精细）+0.01每次按下
- **F**: 切换性能显示（FPS日志）

## 视觉特性

### 黑洞
- **事件视界**：史瓦西半径处的纯黑球体（1.269×10¹⁰ 米）
- **引力透镜**：光在黑洞周围弯曲
- **光子球**：光子轨道的区域，位于1.5×史瓦西半径

### 吸积盘
- **内半径**：2.5×史瓦西半径（~3.17×10¹⁰ 米）
- **外半径**：10×史瓦西半径（~1.269×10¹¹ 米）
- **颜色**：基于温度的黑体辐射
  - 内部：蓝白色（热）
  - 中部：黄白色
  - 外部：橙红色（冷）
- **Shakura-Sunyaev模型（Phase 5）**：
  - 温度分布 T ∝ r⁻³/⁴
  - 光度 L ∝ T⁴
  - 相对论性多普勒增强
  - 引力红移效应

### 天体对象
- 可选的环境天体（恒星、行星）
- 显示为有色球体
- 可选：受牛顿引力影响

### 时空网格
- 显示弯曲时空
- 网格线在黑洞附近弯曲
- 帮助可视化引力效应
- 可以通过代码启用/禁用

## 物理模拟

### 测地线积分
- **方法**：4阶Runge-Kutta（RK4）
- **步长**：D_LAMBDA = 1×10⁷ 米
- **最大步数**：100,000步
- **精度**：能量守恒 ~10⁻⁴

### 守恒量
- **能量（E）**：每单位质量的能量
- **角动量（L）**：每单位质量的角动量
- **Carter常数（Q）**：Kerr度规中的第三守恒量
  - 在Schwarzschild中：Q = 0
  - 在Kerr中：Q ≠ 0（允许更复杂的轨道）

### 引力模拟（启用时）
- **牛顿引力**：F = GMm/r²
- **多体模拟**：所有对象相互影响
- **时间步长**：自适应，基于速度

## 性能

### 自适应分辨率
- **完整质量**：200×150计算分辨率
  - 相机静止时使用
  - 最高视觉质量
  
- **移动质量**：160×120计算分辨率（80%）
  - 相机移动时使用
  - 更快的性能，视觉质量略微降低

### 网格缓存
- 网格仅在对象移动>1×10⁸米时重新生成
- 大幅减少CPU开销
- 性能提升100-1000倍

### HDR管线
- **纹理格式**：RGBA16F（16位浮点）
- **色调映射**：ACES电影
- **泛光**：四分之一分辨率，10次迭代

## 日志系统

项目使用带级别的Logger类：
- **INFO**：常规操作（键按下、参数变化）
- **WARN**：警告（但继续执行）
- **ERROR**：错误（但尝试恢复）
- **DEBUG**：详细输出（网格重新生成、详细诊断）

在代码中更改日志级别：
```cpp
Logger::setLevel(LogLevel::DEBUG);  // 用于详细输出
```

## 技巧

### 探索爱因斯坦环
1. 将相机定位使盘位于黑洞后面
2. 注意光围绕事件视界弯曲
3. 寻找完整的环或弧
4. 使用Shift+点击导出特定射线以供分析

### 观察拖曳效应
1. 启用Kerr度规（K键）
2. 将自旋设置为最大（]键多次）
3. 注意相对于非旋转情况的变化：
   - 内盘边缘移动得更近
   - 一侧比另一侧更亮（多普勒）
   - 拖曳产生的不对称模式

### 性能优化
1. 如果FPS低，减少窗口大小
2. 禁用泛光（B键）以获得更快性能
3. 减少泛光强度（-键）
4. 移动时避免泛光切换

### 科学分析
1. 使用模式3（能量）验证积分精度
2. 使用模式4（Carter）检查Kerr度规守恒
3. 导出射线（P/C）用于MATLAB/Python分析
4. 使用Shift+点击研究特定特征

### 多波段探索（Phase 6）
1. 按W键循环切换不同波段
2. 比较冷外区（射电）与热内区（X射线）
3. 使用多波段模式一次查看所有温度
4. 注意不同波段如何突出不同物理过程
