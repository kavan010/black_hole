## 🎯 第一步：编译程序

### 选项A：使用apt安装依赖（推荐 - Ubuntu/Debian）

```bash
# 1. 安装依赖
sudo apt update
sudo apt install build-essential cmake libglew-dev libglfw3-dev libglm-dev libgl1-mesa-dev

# 2. 编译
cd black_hole
cmake -B build -S .
cmake --build build

# 3. 运行！
./build/BlackHole3D
```

### 选项B：使用vcpkg（跨平台）

```bash
# 1. 克隆项目
git clone https://github.com/kavan010/black_hole.git
cd black_hole

# 2. 安装依赖
vcpkg install

# 3. 配置并编译
vcpkg integrate install
cmake -B build -S . -DCMAKE_TOOLCHAIN_FILE=[vcpkg路径]/scripts/buildsystems/vcpkg.cmake
cmake --build build

# 4. 运行！
./build/BlackHole3D
```

## 🎮 第二步：基本操作

### 相机控制

| 操作 | 效果 |
|------|------|
| 🖱️ **左键拖动** | 围绕黑洞旋转视角 |
| 🖱️ **滚轮** | 放大/缩小 |
| ⌨️ **E键** | 调亮画面 |
| ⌨️ **Q键** | 调暗画面 |
| ⌨️ **R键** | 重置亮度 |

**试一试：** 拖动鼠标，从不同角度观察黑洞！

### 切换黑洞类型

| 键 | 效果 |
|----|------|
| ⌨️ **K键** | 切换旋转/非旋转黑洞 |
| ⌨️ **[ 键** | 降低旋转速度 |
| ⌨️ **] 键** | 提高旋转速度 |

**试一试：**
1. 按K键开启旋转黑洞
2. 按]键多次，提高旋转速度
3. 观察吸积盘的不对称性！

### 特效控制

| 键 | 效果 |
|----|------|
| ⌨️ **B键** | 开关发光效果 |
| ⌨️ **+ 键** | 增强发光 |
| ⌨️ **- 键** | 减弱发光 |

## 🌈 第三步：探索不同视图

### 可视化模式

按 **V键** 循环切换5种视图：

1. **正常模式** - 真实的黑洞外观
2. **红移模式** - 看到引力如何影响光的颜色
3. **步数模式** - 显示计算复杂度
4. **能量模式** - 检查物理守恒
5. **Carter模式** - 验证数值精度

**试一试：** 按V键多次，看看每种模式有什么不同！

### 波段模式（Phase 6新功能！）

按 **W键** 查看不同波段：

1. **射电波段** - 看到冷的外围区域（红橙色）
2. **红外波段** - 看到温暖的尘埃（橙红色）
3. **可见光波段** - 我们眼睛看到的（默认）
4. **X射线波段** - 看到超热的内盘（蓝白色）
5. **多波段** - 同时看到所有温度（彩虹色）

**试一试：** 
1. 按W键切换到X射线模式
2. 注意只有最热的内盘是亮的！
3. 再按W切换到射电模式
4. 现在外围区域更亮了！

## 🔬 第四步：科学探索

### 导出光线路径

想要分析光线如何弯曲？

1. **Shift + 点击** 屏幕上任意位置
2. 程序会保存该光线的完整路径
3. 文件保存为 `ray_click_<时间戳>.csv`

**用途：**
- 用Python绘制光线轨迹
- 分析爱因斯坦环
- 研究光子球

### 导出中心射线

| 键 | 效果 |
|----|------|
| ⌨️ **P键** | 导出视线中心的光线 |
| ⌨️ **C键** | 导出锥形光线束 |

**CSV文件包含：**
- 位置 (x, y, z)
- 速度
- 能量和角动量
- 每一步的详细信息

## 💡 第五步：常见场景

### 场景1：观察爱因斯坦环

```
1. 调整视角，让吸积盘位于黑洞后面
2. 你会看到光围绕黑洞弯曲形成环
3. Shift+点击环的位置
4. 导出的CSV显示光线如何绕过黑洞
```

### 场景2：比较旋转和非旋转黑洞

```
1. 先观察当前（非旋转）黑洞
2. 按K键切换到旋转黑洞
3. 按]键多次增加旋转
4. 注意变化：
   - 内盘边缘移动了
   - 一侧更亮（多普勒效应）
   - 拖曳产生的不对称
```

### 场景3：多波段观测

```
1. 正常观察（可见光）
2. 按W切换到X射线
   → 只看到超热内盘
3. 再按W切换到射电
   → 看到冷外区
4. 再按W切换到多波段
   → 一次看到所有温度！
```

## 🐛 遇到问题？

### 问题：屏幕全黑

**原因1：** 曝光太低
- **解决：** 按E键多次

**原因2：** 太靠近黑洞
- **解决：** 滚轮向后，远离黑洞

### 问题：画面卡顿

**解决方案：**
1. 按B键关闭发光效果
2. 减小窗口大小
3. 按-键降低发光强度

### 问题：编译失败

**缺少依赖：**
```bash
# Ubuntu/Debian
sudo apt install build-essential cmake libglew-dev libglfw3-dev libglm-dev

# Fedora
sudo dnf install gcc-c++ cmake glew-devel glfw-devel glm-devel

# macOS
brew install cmake glew glfw glm
```
